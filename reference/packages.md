# Packages Used

This document lists the primary packages used in this application.

## Core Dependencies

*   **[chrome-launcher](https://www.npmjs.com/package/chrome-launcher):** Launches Google Chrome, which is essential for running Lighthouse audits.
*   **[fs-extra](https://www.npmjs.com/package/fs-extra):** Provides extended file system methods, simplifying file and directory management.
*   **[lighthouse](https://www.npmjs.com/package/lighthouse):** The core of the auditing functionality. It audits web pages for performance, accessibility, SEO, and more.
*   **[playwright](https://www.npmjs.com/package/playwright):** Used for crawling websites and discovering pages to audit. It automates browser interactions.
*   **[yargs](https://www.npmjs.com/package/yargs):** A library for building interactive command-line tools and parsing arguments.
*   **[node-cron](https://www.npmjs.com/package/node-cron):** A simple cron-like job scheduler for Node.js.

## Development Dependencies

*   **[jest](https://jestjs.io/):** A JavaScript testing framework used for writing and running tests.
